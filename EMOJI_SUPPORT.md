## Font Support

ScrollMaster uses **DejaVuSans** font which includes excellent emoji support. The emoji rendering automatically falls back through:

1. **DejaVuSans.ttf** (Primary - best emoji support)
2. **LiberationSans** (if DejaVuSans unavailable)
3. **DejaVuSansMono.ttf** (Limited emoji)
4. **Default bitmap font** (No emoji)

Noto Color Emoji font is also installed as a system font for additional emoji coverage if needed.

## Troubleshooting

If emojis appear as boxes or question marks:
1. The emoji rendering is working, but the Noto Sans font may not have that specific emoji
2. Try using more common emojis (listed above)
3. Make sure the display has sufficient color depth (should be 262K colors)
